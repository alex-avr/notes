# Otus Kotlin Notes

Учебный проект курса [Otus](https://otus.ru) [Kotlin Backend Developer](https://otus.ru/lessons/kotlin/).
Поток курса 2022-12.

Notes -- это простое приложение для заметок.
С его помощью пользователь может создавать свою личную базу знаний,
составленную из зменток по нужным ему темам.
